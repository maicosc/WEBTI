Comandos Básicos do Git

  • git init -> cria um repositório git
  • git status -> visualiza o status do WK e do stage
  • git add[file] -> adiciona as mudanças no stage
  • git commit -m -> faz commit todos os arquivos que estão no stage e mostra uma mensagem
  • git config user.name "name" -> configura o nome a ser usado no commit dos arquivos
  • git config user.email "email" -> configura o email a ser usado no commit dos arquivos

    • OBS: Tanto o user.name, quanto o user.email devem ser, obrigatoriamente, declarados no primeiro commit da máquina. Após isso, não será mais necessário declará-los na mesma máquina

  • git log -> Visualiza o status do repositório
  • git remote add origin [URL do repositório] -> adiciona um repositório remoto
  • git remote -v -> lista todos os repositórios remotos
  • git push [Repositório remoto (origin)] [branch] -> Envia os arquivos do repositório físico para o remoto

    • OBS: Não esqueça de autenticar o token do GitHub

  • git branch -> Visualiza todas as branchs
  • git checkout -b [nome da branch] -> Cria e muda para uma nova branch
  • git checkout [nome da branch] -> abre a branch requerida
  • git clone [URL do repositório] -> clona um repositório
  • git diff -> exibe a diferença entre versões de um repositório
